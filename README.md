# What Should We EAT?
    
## 🔎 About   

오늘 뭐먹지? 🤔 남녀노소 구분없이 매일 한번쯤 하는 고민!   
저희 프로젝트는 아주 평범하고, 일상적인 고민에서 시작했습니다.   

오늘 뭘 먹을까 하는 고민부터    
자신이 그날 먹은 것을 기록하는 ```식단관리``` 기능,   
 나만의 맛집을 추천하고 공유하는 ```맛집추천``` 기능,    
 많은사람과 나만의 레시피를 공유할 수 있는 ```레시피공유``` 기능과 함께 오늘의 식사를 함께하세요!
